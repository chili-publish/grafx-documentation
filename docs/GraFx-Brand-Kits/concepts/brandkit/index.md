# Brand Kit  

A container of [design tokens](/GraFx-Brand-Kits/concepts/elements/) and assets (colors, fonts, media, paragraph - & character styles) that define a visual identity.  

- Defined at the **environment** level

A Brand Kit can carry variations of its visual identity through [themes](/GraFx-Brand-Kits/concepts/themes/): the default theme defines the full brand, and additional themes override only what differs.
