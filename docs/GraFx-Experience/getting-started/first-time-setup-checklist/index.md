# First-Time Setup Checklist

Before you begin using GraFx Experience for production, complete the items below.

## User access & identity

- [ ] Your user account exists (local or federated via SSO).  
- [ ] You have been assigned at least one group with right permissions.
- [ ] You can log in and see the GraFx Experience entry screen.

## Environment readiness

- [ ] At least one **Design System** is published and visible in the GraFx Experience catalog.  
- [ ] Design Systems have variables defined, with clear constraints for end users.  
- [ ] Media assets required by Design Systems are available in **GraFx Media** or via connected DAM.

## Brand configuration

- [ ] **GraFx Brand Kits** are defined (colors, fonts, styles).  
- [ ] Brand assets (logos, icons, approved imagery) are available either in Media or connected DAM.

## Interface & navigation

- [ ] Navigation labels and structure are verified for your users.  
- [ ] Environment settings (like login page branding) are configured according to your organizationâ€™s standards.  
- [ ] User roles and group assignments are reviewed for clarity and least privilege.

## Integration & connectors

- [ ] External DAM (if used) is connected via the **Connector Framework**.  
- [ ] Any required data sources (PIM, CRM, CDP) are connected and passing data correctly.  
- [ ] SSO integration (if used) is active and group attributes map correctly at login.

## Browser & devices

- [ ] Supported browsers are identified and communicated to users.  
- [ ] Devices and resolutions your users will use have been tested.

## Communications

- [ ] User onboarding instructions are prepared and shared.  
- [ ] Support contacts and escalation paths are defined for first-line issues.
