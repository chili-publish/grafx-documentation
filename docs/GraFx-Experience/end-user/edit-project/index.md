# Edit a project

This page explains how to personalize the layouts in your project.

## The editing experience

When you open a project, the editing experience shows:

- **Layout tabs** along the top — one tab for each layout in your project, plus an **All layouts** tab
- **Customize panel** on the left — all editable variables for the selected tab
- **Canvas** on the right — a live preview of the layout(s) updating as you make changes

## Editing all layouts at once

The **All layouts** tab is selected by default. Changes made here apply to every layout in your project simultaneously.

Use this to set values that are consistent across all formats — for example, region, language, tagline, or product.

## Editing a single layout

Click an individual layout tab to switch to that layout. Changes made in this view apply only to that specific layout and do not affect the others.

This is useful when you need to make an exception — for example, a different price point on a specific print format, or a different crop on a social media layout.

## Types of variables

The Customize panel shows all the variables available in the template. Depending on how the template was built in GraFx Studio, these can include:

- **Dropdown (select)** — choose from a predefined list of options (e.g. Region, Language, Product Flavor, Tagline)
- **Text** — enter free text (e.g. a headline or product name)
- **Number** — enter a numeric value (e.g. price)
- **Image** — replace an image by uploading a file or selecting from the media library

When you change a variable, all affected layouts are highlighted in the layout tabs so you can see exactly what will change.

## Adding layouts

You can add more layouts to your project after it has been created. Click the **+** icon in the upper right of the editing experience to open the layout selector and choose additional layouts.

Layouts already in your project are greyed out and cannot be selected again.

## Layout options

Click the **...** context menu next to a layout tab to access additional options:

- **Rename** — give the layout a custom name within this project
- **Duplicate** — create a copy of the layout in the project
- **Remove** — remove the layout from the project

## Saving your work

Click **Save** to save your changes. Your project is stored on the Projects page and can be reopened at any time.

Click **Close** to exit the editing experience without saving the latest changes.

!!! tip
    You can undo and redo changes using the arrow buttons in the toolbar before saving.
