# Download and order output

Once a project is saved, you can produce output — either by downloading a file directly or by sending the project to a production partner.

## Downloading output

From a project card on the Projects page, click the **Print** dropdown button to see the available download options.

The options shown depend on the output intents of the layouts in your project:

- **Download Print** — generates print-ready files for all print layouts in the project
- **Download Digital** — generates digital files for all digital layouts in the project

Downloads are processed in the background. When ready, they appear on the **Downloads** tab of the Projects page.

## Ordering via a production partner

If one or more production channels have been configured for your organization, you can send your project directly to a print partner for fulfillment.

### Step 1: Select a production partner

From the project card, open the **Print** dropdown and select the production partner you want to use.

You can optionally add:

- A desired delivery date (informational — the partner will confirm actual delivery)
- An internal order number
- Additional notes

### Step 2: Enter a delivery address

Enter one or more delivery addresses for the order. You can add multiple addresses if the output needs to be delivered to different locations.

### Step 3: Set quantities

Enter the number of prints you need for each layout. If you entered multiple delivery addresses, you can set different quantities per address.

### Step 4: Review and place the order

Review the full order summary and click **Place order** to send it to the production partner.

## Tracking orders

After placing an order, it appears in your order history (typically accessible from the **Orders** page in the navigation).

From this page you can:

- View the status of each order (Submitted, In Production, Rejected, Delivered)
- See details of individual orders, including layout quantities and delivery addresses
- Cancel an order, as long as it has not yet been picked up by the production partner
- Cancel individual layouts within an order

The production partner receives a notification when an order is placed and can update the status as the order progresses.
