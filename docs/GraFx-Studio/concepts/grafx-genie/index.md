# GraFx Genie for Actions

You need to write an Action, to make your template into a Smart Template. Challenge: you don't know how to write JavaScript code, to make this happen.

GraFx Genie can write [Actions](/GraFx-Studio/guides/actions/create/) for you. Describe the behavior you want in plain language, and GraFx Genie generates the script, using the context of your template. Actions play a vital role within triggers. See also: [Introduction to Actions](/GraFx-Studio/concepts/actions/)
