# Genie for Actions

![rn_icon](/assets/icon-Grafx-Genie.svg){.rn_icon}

## What it is

In GraFx Studio, **Actions** are the JavaScript logic that turns a template into a Smart Template — for example, showing an extra asset when a discount drops below a threshold, or adjusting a layout when a value changes. **Genie for Actions** is the AI assistant that writes that script for you.

## How GraFx Genie helps

Writing Actions normally means writing JavaScript. Genie removes that barrier: you describe the behaviour you want in plain language, and Genie generates the Action script — using the template's context, such as your variable and frame names, so the result fits your template without you spelling everything out. You stay in control: review the suggested script, tweak it, or hand it to a developer for the final touches.
