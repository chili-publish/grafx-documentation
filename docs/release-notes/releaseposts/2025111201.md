### GraFx Studio

- **Introduced HTML output** ([Experimental](/release-notes/experimental/)) for templates.  
  You can now export designs as HTML for use in solutions like digital signage.  
  [See more on HTML output](/GraFx-Studio/guides/output/html/)

- **Added support for frame opacity**.  
  All frames now support the opacity settings (alpha channel), enabling overlay effects and layered layouts.  
  [See more on Opacity](/GraFx-Studio/concepts/frames/#opacity)
  
- **Improved project overview** with new list and grid views.  
    You can now search and sort your existing projects for easier navigation.  
    [See more on My Projects](/GraFx-Studio/concepts/template-management/#my-projects)
    
- **Added** error warning, when template opens with wrongly configured font connector
- **Added** shortcut to Duplicating frames (ALT + Drag)
- **Added** plugin models for gradient

- **Improved** gradient stop removal
- **Optimized** render reporting

- **Fixed** position of action menu for Brand Kit media
- **Fixed** truncation in media panel (title)
- **Fixed** action editor which did not load due frame anchoring loop
- **Fixed** for copyfitting and baseline shift
- **Fixed** infinite anchoring loop
