# Upcoming Change: Superscript Font Features in GraFx Studio

![CHILI GraFx icon](/assets/icon-CHILI-GraFx.svg){.rn_icon}

Some fonts define their own superscript settings — the size and vertical position that superscript characters should take, as specified by the font designer. Until now, GraFx Studio did not apply these custom values and used default superscript positioning instead.

An upcoming bug fix in GraFx Studio **1.46.0** resolves this: superscript text will follow the settings defined in the font, as it already does in GraFx Publisher.

!!! note "Visual changes possible"

    If your templates use fonts with custom superscript features, superscript text may render slightly differently after this update. Only customers using fonts with these features are affected.
