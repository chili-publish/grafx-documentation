# CHILI GraFx Platform API Update

![rn_icon](/assets/icon-CHILI-GraFx.svg){.rn_icon}

## Fixed

### Subscription name now shown correctly in invitation emails

When inviting an existing CHILI GraFx user to a subscription, the invitation email would sometimes show a blank subscription name. This has been fixed — the subscription name now always appears correctly in the email, even when the account's display name is empty.

## New & Improved

### Assign system-level roles via API

A new endpoint `PATCH /api/v1/users/{userId}/system-permissions` allows system administrators to assign support roles to CHILI GraFx users. This enables programmatic management of support access levels without requiring manual intervention.
