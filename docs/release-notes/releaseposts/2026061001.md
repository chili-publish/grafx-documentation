# CHILI GraFx Environment API: Updated Swagger examples and faster responses

![CHILI GraFx icon](/assets/icon-CHILI-GraFx.svg){.rn_icon}

## ✨ New & Improved

### Updated examples for Media and Template endpoints

The Swagger documentation for your environment is the interactive reference for every Environment API endpoint, including example requests and responses. The examples for the Media and Template GET endpoints have been refreshed so they accurately reflect what the API returns today — making it easier to build and debug integrations against these endpoints. See [Making API Calls](/GraFx-Developers/environment-api/04-making-api-calls/) for how to access the Swagger documentation for your environment.

### Faster responses on frequently used operations

The Environment API now communicates more efficiently with its cache on the most heavily used operations, improving the overall responsiveness of the API. Integrations that make frequent calls will see snappier response times without any changes on their side.
