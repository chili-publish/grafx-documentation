# GraFx Publisher Update

![GraFx Publisher icon](/assets/icon-GraFx-Publisher.svg){.rn_icon}

## 🛠️ Fixes & Stability

### Further improvements to font cache memory usage

This update delivers another round of optimizations to how fonts are cached during output generation. Memory consumption for fonts is reduced further, continuing the work started in the previous Publisher release and improving reliability for documents that reference a large number of fonts.
