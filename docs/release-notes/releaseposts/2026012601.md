# Image constraints, list controls, and text stability improvements

![GraFx Studio icon](/assets/icon-GraFx-Studio.svg){.rn_icon} <span class="version-label">v1.37</span>

## GraFx Studio

### ‚ú® New & Improved

- **Added constraints for image frames**  
  You can now constrain resizing, movement, and rotation for image frames.  

  [See: Concept of constraints](/GraFx-Studio/concepts/constraints/)  
  [See: Image frame constraints](/GraFx-Studio/guides/image-frame/#image-frame-constraints)
    
![Constraints](/release-notes/releasenotesassets/constraints.gif){screenshot-full}

- **Added character style support for numbered lists.**  
  Character styles now apply consistently within numbered list items.

- **Added keyboard shortcuts for list level control.**  
  You can increase or decrease levels in bullet and numbered lists using keyboard shortcuts.

- **Added predefined formatting for four numbered list levels.**  
  Numbered lists now follow a fixed, predictable structure across up to four levels.

- **Added enforcement of maximum text variable length.**  
  Text variables now respect defined length limits to prevent overflow and layout breaks.
