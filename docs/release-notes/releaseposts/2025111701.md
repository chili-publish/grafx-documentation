# Centralized Login for GraFx Publisher BackOffice

![rn_icon](/assets/icon-GraFx-Publisher.svg)

Starting **November 16, 2025**, login to the GraFx Publisher BackOffice will be centralized through the CHILI GraFx platform.  
Direct BackOffice URLs will no longer allow authentication.

For details, timelines, and required actions, see [the full update](/release-notes/2025/09/09/upcoming-change-centralized-login-for-grafx-publisher-backoffice/)
