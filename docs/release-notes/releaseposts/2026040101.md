# Components for GraFx Studio

![GraFx Studio icon](/assets/icon-GraFx-Studio.svg){.rn_icon}

## ✨ New & Improved

### Components — build once, reuse everywhere

Components are reusable design elements that can be placed inside templates. Design a pricing block, a product ad, or any branded element once — then place it across as many templates as you need. When you update a component, every template that uses it reflects the change automatically.

**What you can do with components:**

- **Place a component as a frame** — open the Resources panel in the template workspace, browse or search for a component, and click to place it on the canvas
- **Use multiple instances** — place the same component multiple times on a single page, each with its own position, size, and data
- **Map variables per instance** — connect each component instance to its own set of template variables, so a coupon sheet with six coupons can show six different prices from the same pricing component
- **Choose a resize mode** — control how the component fills its frame: Scale (picks the best-matching layout by aspect ratio), Resize (applies internal anchoring and copyfitting rules), or Scale and resize (combines both)

**For template designers:**

Components have their own workspace in GraFx Studio, alongside Templates and Collections. Each component can have multiple layouts, its own Brand Kit, variables, and actions. The component workspace mirrors the template designer workspace, with a focused set of features for reusable design elements.

> Variable mapping is per instance. A template with three instances of the same component can map each to a completely different set of variables — giving each instance its own independent data.

See [Components](/GraFx-Studio/concepts/components/) for a full introduction, including use cases and how components relate to templates.
