# GraFx Publisher Update

![GraFx Publisher icon](/assets/icon-GraFx-Publisher.svg){.rn_icon}

## 🛠️ Fixes & Stability

### Improved stability for image-heavy documents

Resolved out-of-memory exceptions caused by the image cache growing unbounded during output generation. This fix addresses the same class of caching issue previously resolved for fonts, and improves reliability when processing documents that reference a large number of images.

### PDF images now benefit from caching

PDF images used in documents can now be cached during output generation. Previously, a bug prevented PDF images from being stored in the cache, causing them to be re-processed on every output job. This improves performance for documents that reuse PDF images across multiple output runs.
