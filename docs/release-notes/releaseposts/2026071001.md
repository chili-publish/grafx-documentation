# CHILI GraFx Environment API: Large output requests fix

![CHILI GraFx icon](/assets/icon-CHILI-GraFx.svg){.rn_icon}

## 🐛 Bug Fixes

- Fixed `413 Content Too Large` errors on output requests for large documents: the 10MB size limit now applies only to the `variables` property instead of the entire request body, so projects built on big templates download reliably again. See the [Output API limitations](/GraFx-Developers/grafx-studio/supplementary-materials/variable-data-printing-with-output-api/#limitations) for details.
