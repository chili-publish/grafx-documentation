# GraFx Studio Photoshop Plugin 1.2.1

![rn_icon](/assets/icon-GraFx-Studio.svg)

## Fixed

- **Photoshop packages with zero-dimension frames can now be imported successfully.**
  Some Photoshop documents contained frames with a width or height of zero — for example, layers hidden or collapsed in certain layouts. These packages previously failed to import into GraFx Studio with an "Invalid width value" error. They now import correctly.
