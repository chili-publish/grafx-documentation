# GraFx Studio Photoshop Plugin 1.2.1

![rn_icon](/assets/icon-GraFx-Studio.svg){.rn_icon}

## Fixed

### Photoshop packages with hidden frames can now be imported

Photoshop documents containing frames with a width or height of zero — such as frames used for hidden or conditional elements — previously caused the import to fail with an "Invalid width value" error. These packages now import correctly into GraFx Studio.
