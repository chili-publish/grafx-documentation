# CHILI GraFx Environment API Update

![rn_icon](/assets/icon-CHILI-GraFx.svg){.rn_icon}

## New & Improved

### Full subject type overview in a single API call

The `GET /vision?expand=allSubjectTypes` endpoint now returns both legacy top-level properties and all subject types with their individual properties in one response. Developers no longer need multiple calls to get a complete picture of the subject types configured in their environment.

### Fallback text for font previews

When a font preview image is unavailable, the API now returns fallback text instead of an empty result. Font lists in your integrations will display more consistently, even when preview assets are missing.

### Update subject types via PATCH

A new `PATCH /vision?subjectType={subjectId}` endpoint (available on both GraFx and external environments) lets developers update individual subject types directly, without affecting others.
