# Tags for templates in GraFx Studio ✨

![Use tags to find templates faster](/GraFx-Studio/guides/use-tags/tag08.png)

## ✨ New

Added **Tags** for templates.

As your template library grows, Tags help you organize templates by brand, campaign, or product line — so every designer can filter down to what is relevant to them, without relying on naming conventions or scrolling through unrelated content.

Learn more: [Tags concept](/GraFx-Studio/concepts/tags/) — [Manage & Use Tags](/GraFx-Studio/guides/use-tags/)
