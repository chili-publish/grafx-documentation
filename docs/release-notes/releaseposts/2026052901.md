# CHILI GraFx Environment API: Source files preserved in print output

![CHILI GraFx icon](/assets/icon-CHILI-GraFx.svg){.rn_icon}

## 🛠️ Fixes

### JPG and PNG source files are kept for print output

When GraFx Studio generates a PDF, the output engine requests each placed image at full resolution for print. For JPG and PNG assets, the Environment API now returns the original source file, so the image's original color space and quality carry through to the final PDF — a CMYK JPEG stays CMYK all the way to print. This matches the behavior described in the [media connector documentation](/GraFx-Developers/connectors/media-connector/media-connector-fundamentals/#supported-formats).
